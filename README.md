# rfuse-ng

Rfuse NG is a fork of rfuse for Ruby 1.9.2 and above.

## Installation

Add this line to your application's Gemfile:

    gem 'rfuse-ng'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rfuse-ng
